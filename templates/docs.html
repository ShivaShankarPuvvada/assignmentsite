Steps:
  • Has to Create a project first.
  • Create a task. Type and enter the project name and select the project while creating the task.
  • Created tasks can be seen in All tasks.
  • Filtering can be done using from and to dates. Tasks are filtered by created_at date.
  • Countdown counter counts from current time and up to end time of the task.
Backend:
  • Used jwt authentication.
  • Sent jwt through urls instead of sending through headers using client.
  • Used Django default sqlite as database and heroku for hosting.
