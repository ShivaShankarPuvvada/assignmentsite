API:

    Wishlist

  • Create:
    • url: http://legendables.herokuapp.com/port/wishlist/create
    • Accepted methods:
      • GET: Gets the list of all wishlists.
      • POST: Creates a wishlist.
  • Update-delete:
    • url: http://legendables.herokuapp.com/port/wishlists/1
    • Accepted methods:
      • PUT: Updates a wishlist by taking wishlist id.
      • DELETE: Deletes a wishlist by taking wishlist id.
      • GET: Gets the wishlist by taking wishlist id.

    • Container/Folder

  • Create:
    • url: http://legendables.herokuapp.com/port/container/create
    • Accepted methods:
      • GET: Gets the list of all containers.
      • POST: Creates a container.
  • Update-delete:
    • url: http://legendables.herokuapp.com/port/containers/1
    • Accepted methods:
      • PUT: Updates a container by taking container id.
      • DELETE: Deletes a container by taking container id.
      • GET: Gets the container by taking container id.

    • Video/Movie

  • Create:
    • url: http://legendables.herokuapp.com/port/movie/create
    • Accepted methods:
      • GET: Gets the list of all movies.
      • POST: Creates a movie.
  • Update-delete:
    • url: http://legendables.herokuapp.com/port/movies/1
    • Accepted methods:
      • PUT: Updates a movie by taking movie id.
      • DELETE: Deletes a movie by taking movie id.
      • GET: Gets the movie by taking movie id.
How to:

    Firstly, create a wishlist

  • Data: {"name": "sample wishlist name"}
  • Create Url: http://legendables.herokuapp.com/port/wishlist/create
  • Update-delete Url: http://legendables.herokuapp.com/port/wishlists/CREATED WISHLIST ID(ex: 1,2,3 etc.., )

    • Secondly, create a folder

  • Data: {"wishlist": CREATED WISHLIST ID(ex: 1,2,3 etc.., ), "name": "sample folder name"}
  • Create Url: http://legendables.herokuapp.com/port/contianer/create
  • Update-delete Url: http://legendables.herokuapp.com/port/containers/CREATED CONTAINER ID(ex: 1,2,3 etc.., )

    • Third, create a movie

  • Data: {"wishlist": CREATED WISHLIST ID(ex: 1,2,3 etc.., ), "folder": CREATED CONTAINER ID(ex: 1,2,3 etc.., ), "title": "doctor who", "url": "https://www.google.com", "released_date": "2012-09-11"} (Note: Folder field is optional, it depends on if the video want to be saved in folder or not, if you want to save the video in the folder you have to send the folder/container id. if you dont want to save it in the folder and directly want to save in the wishlist then "folder" is not needed)
  • Create Url: http://legendables.herokuapp.com/port/movie/create
  • Update-delete Url: http://legendables.herokuapp.com/port/movies/CREATED MOVIE ID(ex: 1,2,3 etc.., )
